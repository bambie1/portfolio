A notes-taking web app

Travel Diary

A NodeJS app for journalling your upcoming trips. The user enters a trip details (location and date), and a trip card with useful information (weather and location-photo) is dynamically created with async Javascript and multiple API calls. All trip cards are saved on the server along with user-notes. The HTML, Sass and JS files were minified and built with Webpack.
